SupplySync API

SupplySync API is a powerful tool designed to help businesses streamline their supply chain management and ensure smooth operations. The API allows for easy integration with various systems, providing real-time data on inventory levels, order processing, and supplier relationships.

Features:
•	Inventory Management: Track stock levels, automate restocking, and avoid stockouts.
•	Order Processing: Seamlessly manage incoming and outgoing orders, with real-time status updates.
•	Supplier Integration: Sync data with suppliers for efficient procurement and logistics.
•	Analytics & Reporting: Get insights into inventory performance, order trends, and more.

Technologies Used:

•	Java & HTML – Leveraged for backend development and crafting responsive web interfaces, ensuring a seamless user experience.

•	Behavior-Driven Development (BDD) with Cucumber & Rest Assured – Implemented BDD practices for automated API testing, ensuring robust and reliable API functionality.

•	MySQL – Utilized as a relational database solution for efficient storage, management, and retrieval of inventory and order data.
